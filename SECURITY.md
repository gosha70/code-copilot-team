# Security Policy

## Supported Versions

Security fixes are provided for the latest state of the default branch.

| Version | Supported |
| ------- | --------- |
| `master` | :white_check_mark: |

## Reporting a Vulnerability

Please do not open public issues for security vulnerabilities.

Use GitHub Security Advisories to report privately:

1. Open the repository Security tab.
2. Select "Report a vulnerability."
3. Provide reproduction details, impact, and suggested mitigations.
4. Or use the direct submission link:
   https://github.com/gosha70/code-copilot-team/security/advisories/new

You can expect an initial response within 5 business days.
Validated reports will be addressed as quickly as possible.
